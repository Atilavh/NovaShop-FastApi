# NovaShop-FastApi
NovaShop is a modern e-commerce built with FastAPI  designed to simulate a real-world multi-vendor marketplace system.  This project includes user management, product handling, orders, and shopping cart functionality, focusing on clean API design, scalable architecture, and real-world backend scenarios.

🚀 Features

JWT Authentication system

Role-based access control (Customer / Seller / Admin)

Product & category management

Shopping cart system

Order management with full lifecycle

Concurrency handling to prevent overselling

Pagination and filtering

Redis caching (optional)

Background tasks for async operations

Dockerized environment

Modular and scalable architecture

🧠 Project Goal
The goal of this project is to practice real-world backend engineering concepts, including:

Designing production-like REST APIs
Handling database transactions safely
Optimizing database queries
Working with caching and background jobs
Building scalable and maintainable architecture

⚙️ Tech Stack

Python
FastAPI
PostgreSQL
Redis
Docker

📌 Note
This is not a simple tutorial project. It is designed to reflect real-world backend system design and production-level considerations.
